# make-app
# 出席確認アプリ

学籍番号と学生証の番号をシステムに関連付け、授業時間割を表示するアプリです。

## 主な機能
- 学籍番号・学生証番号の入力とバリデーション
- 月曜〜土曜、1〜6限の授業時間割の自動表示
- レスポンシブデザイン対応
